# Ejemplo de configuración para claude_desktop_config.json

## Versión 2.0 - Configuración simplificada

Solo necesitas agregar UNA ÚNICA entrada para el MCP de SQL Server:

```json
{
  "mcpServers": {
    "sqlserver": {
      "command": "node",
      "args": ["C:\\mcp-sqlserver\\index.js"]
    }
  }
}
```

## Versión anterior (1.0) - NO USAR

Antes necesitabas una entrada por cada conexión:

```json
{
  "mcpServers": {
    "sqlserver-localhost": {
      "command": "node",
      "args": ["C:\\mcp-sqlserver\\index.js"],
      "env": {
        "MSSQL_SERVER": "localhost",
        "MSSQL_DATABASE": "2Personnel",
        "MSSQL_USER": "sa",
        "MSSQL_PASSWORD": "password",
        "MSSQL_PORT": "1433",
        "MSSQL_ENCRYPT": "false",
        "MSSQL_TRUST_CERT": "true"
      }
    },
    "sqlserver-server2": {
      "command": "node",
      "args": ["C:\\mcp-sqlserver\\index.js"],
      "env": {
        // ... más configuración
      }
    }
  }
}
```

## Ventajas de la nueva versión

✅ Una sola entrada en claude_desktop_config.json
✅ Todas las conexiones centralizadas en connections.json
✅ Fácil agregar/modificar conexiones sin reiniciar Claude Desktop
✅ Organización por grupos (cliente, proyecto, etc.)
✅ Descripciones claras de cada conexión
✅ Metadata en todas las respuestas

## Migración

1. Crea connections.json con todas tus conexiones
2. Reemplaza todas las entradas "sqlserver-xxx" por una sola entrada "sqlserver"
3. Reinicia Claude Desktop
4. ¡Listo!

Consulta el README.md para más detalles.
